# VELA Creative Agency

A high-end, conceptual landing page for a fictional Mumbai-based creative studio. Built purely with unadulterated web technologies, this project rejects standard algorithmic templates in favor of physical, human-centric, and asymmetrical design logic.

[![Live Demo](https://img.shields.io/badge/Live_Demo-View_Site-000000?style=for-the-badge)](https://vrchampion.github.io/VELA-Creative-Agency/)

## Features
- **Immersive Environment:** Fullscreen video hero background anchoring the initial scroll.
- **Dynamic Content:** Infinite animated services ticker and horizontal scroll case study reel (featuring Rasa, Vistaar, and Thar Collective).
- **Physical Context:** Real-time data streams displaying a live IST clock and active Mumbai weather directly in the footer.
- **Interactive Forms:** Multi-step modal architecture for project booking (service selection → date/time picker → contact form → confirmation).
- **Embedded Capabilities:** Context-aware "Ask anything" AI chat widget section.
- **Kinetic Typography:** Custom scroll-velocity algorithms mapping physical scroll speed to headline distortion.
- **Attention Context:** Custom idle state that mimics signal decay when the user stops interacting with the site.
- **Content Hierarchy:** 8 distinct asymmetrical sections (Hero, Work, Services, Process, Testimonial, About/Studio, Footer).

## Tech Stack
Built entirely without build steps or modern bloat to guarantee maximum execution speed and total aesthetic control.
- **Markup:** HTML5
- **Styling:** CSS3 (Vanilla, native custom properties and media queries)
- **Logic:** Vanilla JavaScript (No libraries, zero dependencies)
- **Deployment:** GitHub Pages

## Project Structure
```text
/
├── index.html    # Core semantic structure
├── style.css     # Design tokens and all aesthetic logic
└── main.js       # Real-time state management, interaction loops, and API fetching
```

## Running Locally
Because this project utilizes zero build tools, running it locally requires no package managers or server installation. 
To view the site:
1. Clone the repository.
2. Open `index.html` directly in any modern web browser.
